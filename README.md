# ccf_crawler

HELP-DOC:《中国计算机学会通讯》下载工具使用教程.pdf

中国计算机学会通讯，每个月都会发布20余篇文章，这些文章大多关于特邀报告，专栏，技术动态，译文等。

如果手动下载这些通讯的话，有两个问题：

（1）如果逐个下载的话，要逐个点击到相应页面，然后再点击“下载”按钮；

（2）下载下来的文件又都是序号命名，例如5.pdf，11.pdf等，看着一堆数字文件名，浏览起来很是费劲；

这个爬虫程序的主要目的就是解决上面两个问题。

只需要提供一个期刊的id，爬虫程序就会自动把这一期期刊中的文章下载到本地，而且文件命名也和网页上看到的名称一样。

具体使用方法，参考：《中国计算机学会通讯》下载工具使用教程.pdf
